Pandas basics - Analysing tabulated and observational datasets¶

Pandas is a Python library for analysing tabular data. If you've used the Xarray library with gridded datasets you'll find a lot in common with Pandas - Xarray was heavily based upon some of the ideas in Pandas.

One of the main sources of tabulated datasets are observations - you might for instance have a CSV file containing weather station measurements at different times and stations (what I'll be looking at here) or you could be getting data from spreadsheets or SQL databases, all of which Pandas can read from directly. If your data is in a NetCDF file you can also load it with Xarray and convert to Pandas from there.

Let's start out by loading a simple little dataset. I'm reading from a string here for demonstration purposes, normally you'd just give pandas.read_csv() the path to your csv file.

In [1]:
%matplotlib inline
import io
import pandas

# Our observations
observations = """
time,station id,temperature,relative humidity
20181130T0000,001,26,87
20181130T0000,002,30,63
20181130T0100,001,28.5,76
"""

# Create a dummy file with the string `observations` as its contents
# (If you have a real file just give the path to `pandas.read_csv()`)
csv = io.StringIO(observations)

# Read the file
df = pandas.read_csv(csv)
df
Out[1]:
time station id temperature relative humidity
0 20181130T0000 1 26.0 87
1 20181130T0000 2 30.0 63
2 20181130T0100 1 28.5 76

pandas.read_csv() returns a pandas.DataFrame - basically a table of data, with labelled rows and columns. Jupyter displays these tables nicely if you use notebooks to analyse data.

Selecting Columns¶

You can select single columns from a dataframe using dictionary syntax. Pandas calls a single column a pandas.Series. These work like 1d numpy arrays, and most functions that work on a numpy.array will work on a pandas.Series.

In [2]:
# Returns a pandas.Series
print(df['temperature'])

df['temperature'].mean()

0    26.0
1    30.0
2    28.5
Name: temperature, dtype: float64
Out[2]:
28.166666666666668

Another way to select columns is to pass a Python list of column names in the square brackets. This will return a new DataFrame containing only the listed columns

In [3]:
# Returns a pandas.DataFrame
df[['temperature']]
Out[3]:
temperature
0 26.0
1 30.0
2 28.5

Selecting Rows¶

You can select a row from a DataFrame using either .loc[] or .iloc[]. .iloc[] will select rows by row number (just like a numpy array you can also use ranges)

In [4]:
# Single row
df.iloc[1]
Out[4]:
time                 20181130T0000
station id                       2
temperature                     30
relative humidity               63
Name: 1, dtype: object
In [5]:
# Multiple rows
df.iloc[1:2]
Out[5]:
time station id temperature relative humidity
1 20181130T0000 2 30.0 63

.loc[] on the other hand will filter a selection using what's called an index. This is a special column in the DataFrame that's optimised for quick searches.

You can specify an index using the .set_index() function, which returns a new DataFrame with the given column as the index.

In [6]:
df_time = df.set_index('time')
df_time
Out[6]:
station id temperature relative humidity
time
20181130T0000 1 26.0 87
20181130T0000 2 30.0 63
20181130T0100 1 28.5 76
In [7]:
# N.B. - the 'time' column is a string here, not a Python date
df_time.loc['20181130T0000']
Out[7]:
station id temperature relative humidity
time
20181130T0000 1 26.0 87
20181130T0000 2 30.0 63

Like Xarray, dates in Pandas are a bit of a special case, and have extra features by sorting by year, month, day etc. To get these special features you need to tell Pandas to interpret that column as a date, instead of just a normal string. There are other ways to do this, but the simplest way is just to list date columns in the parse_dates parameter when you open the CSV file.

In [8]:
# Read the same CSV file, but use column 0 (time) as the index and convert column 0 to a Python date
csv = io.StringIO(observations)
df_time = pandas.read_csv(csv, index_col=0, parse_dates=[0])
df_time
Out[8]:
station id temperature relative humidity
time
2018-11-30 00:00:00 1 26.0 87
2018-11-30 00:00:00 2 30.0 63
2018-11-30 01:00:00 1 28.5 76
In [9]:
# When the index is a Python date it has some special features, here we find all values on the same day
df_time.loc['20181130']
Out[9]:
station id temperature relative humidity
time
2018-11-30 00:00:00 1 26.0 87
2018-11-30 00:00:00 2 30.0 63
2018-11-30 01:00:00 1 28.5 76

You can also use .loc[] to filter by both row and column at the same time:

In [10]:
# Filtering by both row and column
print(df_time.loc['20181130T0000', 'temperature'])
print('\nMean: ', df_time.loc['20181130T0000', 'temperature'].mean())

time
2018-11-30    26.0
2018-11-30    30.0
Name: temperature, dtype: float64

Mean:  28.0

Filtering rows¶

Our DataFrame contains the measurements from several observation stations. How can we look at individual station measurements, and compare the different stations against each other?

One way to do this is to use a boolean index - first create a series that is True for the rows we're interested in, and False for the rows we don't want, then use that series as an index like so:

In [11]:
# Only station id 001
df_time[df_time['station id'] == 1]
Out[11]:
station id temperature relative humidity
time
2018-11-30 00:00:00 1 26.0 87
2018-11-30 01:00:00 1 28.5 76

Another way is to use something called a pivot table. You may have come across this idea in spreadsheet programs, it splits up a single 1d data column into a 2d table, by using two other columns to label what row and column each value should go into.

Here we want to use 'temperature' as the data source and 'station id' to label the columns - so station 001 is one column, station 002 is the next and so on. For the rows we want to keep the current time index:

In [12]:
station_temp = df_time.pivot(columns='station id', values='temperature')
station_temp
Out[12]:
station id 1 2
time
2018-11-30 00:00:00 26.0 30.0
2018-11-30 01:00:00 28.5 NaN

Note that where we don't have data, in this case station 002 at 1 am, Pandas has put NaN to mark that table cell as invalid.

Once we have the pivot table it's super-simple to create a plot to compare the stations just with .plot(), which makes a quick plot using all columns in the DataFrame:

In [13]:
station_temp.plot(style='D-')
Out[13]:
<matplotlib.axes._subplots.AxesSubplot at 0x7fc9d1c9c5f8>
